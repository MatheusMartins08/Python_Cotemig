-- Calculadora em Python

Uma calculadora simples desenvolvida em Python, com suporte a múltiplas operações matemáticas
e tratamento de erros para garantir maior segurança e estabilidade durante a execução.

-- Funcionalidades
Soma
Subtração
Multiplicação
Divisão (com tratamento para divisão por zero)
Potência
Raiz quadrada (com validação para números negativos)
Execução contínua (loop infinito até o usuário decidir sair)
Tratamento de erros para entradas inválidas

-- Tecnologias utilizadas
Python 3.x
Biblioteca padrão (math)

-- Como usar

Ao executar o programa, você deverá:

Inserir dois números
Escolher a operação desejada no menu:
0 - Sair
1 - Soma
2 - Subtração
3 - Multiplicação
4 - Divisão
5 - Potência
6 - Raiz quadrada
O programa exibirá o resultado e continuará rodando até que você escolha a opção 0 (Sair).
